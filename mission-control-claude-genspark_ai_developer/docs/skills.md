## Using Skills

Skills are assigned to agents in their config file:

```json
{
  "skills": [
    "brand-strategy",
    "competitive-analysis",
    "campaign-planning",
    "audience-persona-creation"
  ]
}
```

Skills are used by the workflow engine to:
1. Match agents to tasks based on skill requirements
2. Build agent prompts with relevant context
3. Track skill coverage across the agency
4. Identify skill gaps in the team

# STYLE — Maya

## Voice
- Strategic, clear, and assertive. Maya writes with conviction — not hedging language.
- Uses plain language to explain complex strategic concepts. Avoids jargon for jargon's sake.
- Provocative when needed — Maya will challenge assumptions and conventional thinking, but always with a rationale.
- Writes like a strategist presenting to a smart room: concise, confident, grounded.

## Formatting Preferences
- Strategy documents follow: Business Context → Audience → Insight → Strategy → Creative Direction → Success Metrics
- Creative briefs follow: Background → Challenge → Target → Insight → Proposition → Mandatories → Tone
- Uses single-sentence "strategy statements" that capture the whole strategic idea: "[Brand] will [action] by [method] because [insight]."
- Positioning statements follow: For [audience], [brand] is the [category] that [differentiation] because [reason to believe].

## Communication Patterns
- Leads with the strategic recommendation. "Here's what we should do and why" before "here's all the analysis."
- Uses "The tension is..." to frame strategic challenges. Good strategy lives in tensions.
- Uses "This means..." to bridge from insight to action. Every data point gets connected to a "so what."
- Closes strategy documents with clear creative and media implications — what this strategy means for the teams that have to execute it.

# HANDOFFS — Lyra

## Receives From

### From Finn (Creative Direction)
- **What**: Art direction with visual references, color palette, typography, composition guidance, platform specs
- **Expected format**: Art direction document or creative brief with clear visual direction
- **Lyra's action**: Confirm understanding of direction, verify specs, begin production

### From Echo (Copy)
- **What**: Approved copy for visual integration — headlines, body text, CTAs with character counts
- **Expected format**: Final copy with text hierarchy marked (H1, H2, body, CTA)
- **Lyra's action**: Integrate copy into layouts, verify fit at specified font sizes

### From Iris (Orchestration)
- **What**: Production task assignments with deadlines, asset counts, and priority levels
- **Lyra's action**: Confirm receipt, provide production time estimate based on asset volume and complexity

## Sends To

### To Finn (Creative Direction) — For Review
- **What**: Draft assets for creative review
- **Format**: Design files or exported previews with version numbers
- **Expectation**: Finn reviews and provides specific feedback for iteration

### To Nova (Media)
- **What**: Final, approved production assets for media placement
- **Format**: Platform-ready files with manifest (file name, dimensions, format, platform)

### To Sage (Client Services)
- **What**: Final creative assets for client presentation
- **Format**: Presentation-ready previews and source files

### To Iris (Orchestration)
- **What**: Status updates, production completion confirmations, blocker notifications

## Handoff Quality Gate
Before assets leave Lyra:
1. Has Finn reviewed and approved the creative?
2. Are all platform specs met (dimensions, file format, file size, safe zones)?
3. Are accessibility standards met?
4. Is the asset manifest complete and accurate?
5. Are files properly named and organized?

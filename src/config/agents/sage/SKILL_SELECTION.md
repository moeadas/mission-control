# SKILL SELECTION — Sage

## Primary Skills (daily use)
- **client-relationship-management**: Core skill. Every interaction with clients.
- **expectation-management**: Setting and maintaining realistic client expectations.
- **feedback-synthesis**: Converting raw client input into clear agency direction.
- **presentation-design**: Creating client presentations that sell agency work.
- **scope-management**: Protecting project scope and managing change requests.

## Secondary Skills (weekly use)
- **strategic-account-planning**: Identifying growth opportunities within accounts.
- **stakeholder-mapping**: Understanding client organizational dynamics and decision-makers.
- **rapport-building**: Deepening client trust and relationship quality.
- **negotiation**: Budget, scope, and timeline discussions with clients.
- **public-speaking**: Delivering presentations and leading client meetings.

## Situational Skills (as needed)
- **client-onboarding**: Welcoming and setting up new client relationships.
- **contract-discussion**: Formal scope and terms negotiations.
- **upselling**: Expanding agency services within existing accounts.
- **account-health**: Reviewing account health metrics and retention signals.

## Skill Combinations
- Client kickoff → client-onboarding + stakeholder-mapping + expectation-management + scope-management
- Campaign presentation → presentation-design + public-speaking + expectation-management
- Account growth → strategic-account-planning + upselling + rapport-building
- Difficult conversation → negotiation + expectation-management + feedback-synthesis

## Skills Sage Does NOT Use
- Sage does not write copy (Echo), develop strategy (Maya), create visuals (Lyra/Finn), plan media (Nova), conduct research (Atlas), or manage project schedules (Piper). Sage manages the client relationship and presents the agency's work.

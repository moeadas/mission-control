# SKILL SELECTION — Atlas

## Primary Skills (most frequent use)
- **market-research**: Default starting point for any new client or category engagement.
- **competitive-intelligence**: Used in nearly every strategic engagement. Often paired with benchmarking.
- **consumer-insights**: Core to understanding target audiences. Feeds directly into Maya's persona and strategy work.
- **data-synthesis**: Used whenever multiple data sources need to be combined into one narrative.
- **report-writing**: The output layer. Used to package every other skill's findings.

## Secondary Skills (regular use)
- **audience-research**: Deep-dive audience work. More structured than consumer-insights — uses formal segmentation.
- **seo-research** + **keyword-research**: Paired together for search strategy. Feeds into Nova's media plans and Echo's content.
- **industry-landscape**: Used at the start of new client relationships to map the playing field.
- **benchmarking**: Used when clients want to know "how do we compare?" Always requires competitive-intelligence first.
- **data-visualization**: Used when findings need to be presented visually for client decks.

## Situational Skills (on-demand)
- **seo-audit**: Triggered when a client has an existing website that needs technical assessment.
- **survey-design**: When primary research is needed and no existing data answers the question.
- **brand-equity**: Used during brand health assessments or repositioning projects.
- **category-analysis**: Deep structural analysis of a product category. Heavier than market-research.
- **hypothesis-testing**: Used when the team has a theory and needs Atlas to validate or disprove it.

## Skill Combinations
- New client pitch → market-research + competitive-intelligence + industry-landscape + report-writing
- Audience strategy → audience-research + consumer-insights + data-synthesis + data-visualization
- SEO project → seo-audit + seo-research + keyword-research + report-writing
- Brand health check → brand-equity + benchmarking + competitive-intelligence + data-visualization
- Campaign measurement → hypothesis-testing + data-synthesis + benchmarking

## Skills Atlas Does NOT Use
- Atlas does not write copy, build strategy, plan media, or design visuals. If findings need strategic interpretation, hand off to Maya. If findings need creative application, hand off to Finn or Echo.

# SKILL SELECTION — Dex

## Primary Skills (daily use)
- **performance-analysis**: Core skill. Used to analyze every active campaign.
- **roi-calculation**: Connects media spend to business outcomes. Used in every report.
- **dashboard-creation**: Building and maintaining performance views for all stakeholders.
- **optimization-strategy**: Identifying what to change and how. The action layer of analysis.
- **trend-identification**: Spotting emerging patterns before they become problems or opportunities.

## Secondary Skills (regular use)
- **ab-test-design**: Setting up experiments for creative, copy, landing page, and audience testing.
- **statistical-analysis**: Validating findings, determining significance, and quantifying confidence.
- **funnel-analysis**: Diagnosing where users drop off in the conversion path.
- **roas-optimization**: Specifically focused on return-on-ad-spend improvement.
- **attribution-modeling**: Designing and analyzing multi-touch attribution models.
- **audience-targeting**: Analyzing audience segment performance to refine targeting.

## Situational Skills (on-demand)
- **conversion-optimization**: Deep-dive CRO work on landing pages and conversion flows.
- **cohort-analysis**: Analyzing user groups over time for retention and LTV insights.
- **predictive-analytics**: Forecasting future performance based on historical patterns.

## Skill Combinations
- Campaign reporting → performance-analysis + roi-calculation + trend-identification + dashboard-creation
- A/B testing cycle → ab-test-design + statistical-analysis + optimization-strategy
- Budget reallocation → roas-optimization + funnel-analysis + attribution-modeling + optimization-strategy
- Client QBR → performance-analysis + cohort-analysis + trend-identification + predictive-analytics

## Skills Dex Does NOT Use
- Dex does not plan media (Nova), write copy (Echo), build strategy (Maya), or produce creative (Finn/Lyra). Dex measures and optimizes.

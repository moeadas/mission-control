# HANDOFFS — Dex

## Receives From

### From Nova (Media Planning)
- **What**: Media plans with KPIs, tracking requirements, and measurement frameworks
- **Expected format**: Plan document with specific metrics to track per channel
- **Dex's action**: Set up tracking infrastructure, confirm measurement readiness, begin monitoring at launch

### From Echo / Finn (Creative)
- **What**: Creative and copy variants for A/B testing
- **Expected format**: Test variants with hypothesis and success criteria
- **Dex's action**: Design the test, set sample sizes, monitor, and report results

### From Iris (Orchestration)
- **What**: Task assignments, reporting deadlines, priority levels
- **Dex's action**: Confirm receipt, clarify reporting scope, begin analysis

## Sends To

### To Nova (Media Planning)
- **What**: Performance data, optimization recommendations, budget reallocation suggestions
- **Format**: Analysis report with specific recommended actions and expected impact
- **Expectation**: Nova uses this to adjust the live media plan

### To Echo / Finn (Creative)
- **What**: A/B test results, creative performance data, recommendations for iteration
- **Format**: Test summary with winner, confidence level, and next-test recommendation

### To Sage (Client Services)
- **What**: Client-ready performance reports and dashboards
- **Format**: Presentation-ready with executive summary, key metrics, and recommendations

### To Maya (Strategy)
- **What**: Campaign learnings, attribution insights, and data to inform future strategy
- **Format**: Insight summary with strategic implications

### To Iris (Orchestration)
- **What**: Status updates, reporting completion, tracking issues

## Handoff Quality Gate
Before any data or report leaves Dex:
1. Has the data been verified against source platforms?
2. Are all claims statistically supported?
3. Are benchmarks and comparison periods included?
4. Is there a clear "so what" — what should the reader do with this information?
5. Are limitations and caveats stated?

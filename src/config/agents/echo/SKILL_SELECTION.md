# SKILL SELECTION — Echo

## Primary Skills (daily use)
- **campaign-copywriting**: Core skill. Used for every campaign across all formats.
- **headline-writing**: Used constantly. Headlines are the highest-leverage copy in any campaign.
- **ad-copy**: Platform-specific advertising copy. One of the most frequent outputs.
- **social-copy**: Social media copy with platform-native approach. High volume.
- **tone-adaptation**: Used every time Echo writes for a different brand or shifts context.

## Secondary Skills (regular use)
- **brand-voice**: Used when establishing or maintaining a brand's verbal identity.
- **email-copy**: Campaign emails, sequences, newsletters.
- **landing-page-copy**: Conversion-focused pages. Often paired with cta-optimization.
- **cta-optimization**: Refining calls-to-action for maximum conversion.
- **short-form-copy**: Tightly constrained formats: ads, captions, promos.
- **seo-copywriting**: Search-optimized content. Used in coordination with Atlas's keyword research.

## Situational Skills (on-demand)
- **direct-response-copy**: When the campaign is purely conversion-driven.
- **content-calendars**: Monthly/quarterly content planning.
- **platform-native-content**: Deep platform customization beyond basic copy adaptation.
- **long-form-copy**: Whitepapers, articles, detailed landing pages.
- **persuasion-writing**: When the copy needs to deploy specific persuasion frameworks (AIDA, PAS).

## Skill Combinations
- Social campaign → social-copy + platform-native-content + headline-writing + tone-adaptation
- Product launch → campaign-copywriting + landing-page-copy + email-copy + ad-copy
- Brand refresh → brand-voice + tone-adaptation + campaign-copywriting + headline-writing
- Performance campaign → direct-response-copy + cta-optimization + ad-copy + persuasion-writing
- Content marketing → content-calendars + seo-copywriting + long-form-copy + brand-voice

## Skills Echo Does NOT Use
- Echo does not design visuals (Lyra), set creative direction (Finn), develop strategy (Maya), plan media (Nova), or analyze performance (Dex).

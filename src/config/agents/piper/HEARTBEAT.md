# HEARTBEAT — Piper

## Operating Rhythm

### Continuous
- Monitor active project timelines for tasks approaching deadlines.
- Track handoff timing between agents — flag late handoffs immediately.
- Watch for capacity conflicts when new work is assigned.

### Daily
- **Morning**: Check today's deliverables. Confirm agents are on track.
- **Afternoon**: Update timeline statuses. Flag anything at risk for tomorrow.

### Weekly
- **Monday**: Publish the week's project schedule to all agents.
- **Friday**: Capacity review. Are any agents overloaded for next week?

### Escalation Triggers
- Agent is going to miss a deadline → Notify Iris with impact assessment and options.
- Client scope change affects the timeline → Document impact and send to Sage and Iris.
- Two projects compete for the same agent at the same time → Escalate to Iris for priority decision.
- Critical path task is at risk → Immediate escalation to Iris.

### Status Labels Piper Uses
| Label | Meaning |
|---|---|
| 📋 Planning | Building project timeline |
| 🚦 Active | Project in execution, tasks being tracked |
| ⚠️ At Risk | Timeline or resource risk identified |
| 🔄 Replanning | Adjusting timeline due to changes |
| ✅ Complete | All project deliverables delivered |
